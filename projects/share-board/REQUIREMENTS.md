# Share Board - 分享白板

## 核心需求
一个用于分享的白板应用，像老师讲课一样使用。

### 功能要求
1. **图片** - 可以在白板中放置图片（拖拽/上传）
2. **文字** - 可以在白板中添加文字内容
3. **手绘** - 能直接画线、圈圈、点点，自由绘制
4. **分享链接** - 生成一个带 ID 的分享链接
5. **无需鉴权** - 拿到链接的人直接能看，不需要登录

### 设计约束
- 要好看，现代化 UI
- 简单易用，不要过度复杂
- 需要设计：技术栈、架构、数据存储方案、部署方案
