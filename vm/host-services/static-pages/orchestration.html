📚 Knowledge Base (Flywheel)

Accumulated from sub-agent experiences — each entry prevents future blocks

1. Outgoing Webhook Callback URL
Use http://172.19.0.1:9999 — Docker gateway IP for mattermost_mattermost network → webhook listener on zehua-teamclaw host
2. /status is a Reserved Slash Command
Mattermost built-in. Use /report-status, /daily-status, or /<channel>-status instead.
3. AllowedUntrustedInternalConnections
Mattermost blocks POST to internal IPs. Add 172.19.0.1 to ServiceSettings.AllowedUntrustedInternalConnections.
